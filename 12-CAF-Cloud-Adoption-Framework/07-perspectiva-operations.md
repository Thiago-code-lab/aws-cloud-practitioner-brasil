# 🛠️ Perspectiva Operations

## 📌 O que é?
É a perspectiva que garante operação confiável e contínua dos workloads, incluindo observabilidade, incidentes, mudanças e continuidade.

## 🧠 Analogia Simples
É o centro de operações de uma concessionária: monitora, corrige e evita parada do serviço.

## 🏗️ Como funciona
Inclui Observability, Event Management, Incident & Problem Management e Patch/Configuration Management.

```text
Métricas/Logs -> Alerta -> Resposta operacional -> Aprendizado e melhoria
```

## 🎯 Casos de Uso Comuns
- Operação 24x7 com métricas e alertas
- Gestão de incidentes e mudanças
- Automação de patching e execução operacional

## 💰 Modelo de Preço
Custos de monitoramento, armazenamento de logs e ferramentas operacionais.

## ⚙️ Comandos CLI Úteis
```bash
aws cloudwatch describe-alarms
aws ssm describe-instance-information
```
