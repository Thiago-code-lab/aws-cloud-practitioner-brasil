# 🧭 Visão Geral do CAF

## 📌 O que é?
É a visão macro do AWS CAF: uma abordagem para transformar negócio e tecnologia com foco em valor, governança e escala.

## 🧠 Analogia Simples
É como um plano de mudança de fábrica: você define objetivo, organiza equipes, executa pilotos e escala com padrão.

## 🏗️ Como funciona
CAF organiza a jornada em quatro fases:

```text
1) Envision: definir objetivos de negócio
2) Align: identificar lacunas de capacidade
3) Launch: executar pilotos com governança
4) Scale: expandir e padronizar em escala
```

## 🎯 Casos de Uso Comuns
- Criar roadmap cloud executivo
- Alinhar TI, segurança e finanças
- Definir metas de maturidade cloud

## 💰 Modelo de Preço
Sem cobrança do framework; custos aparecem na execução do plano.

## ⚙️ Comandos CLI Úteis
```bash
aws organizations list-roots
aws controltower list-enabled-controls --target-identifier ou-xxxx-xxxxxxxx
```
