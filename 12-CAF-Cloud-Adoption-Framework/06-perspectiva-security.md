# 🔐 Perspectiva Security

## 📌 O que é?
É a perspectiva que define controles para identidade, proteção de dados, detecção de ameaças e resposta a incidentes.

## 🧠 Analogia Simples
É o sistema de segurança de um aeroporto: controle de acesso, monitoramento constante e protocolo de resposta.

## 🏗️ Como funciona
Inclui Security Governance, IAM, Threat Detection, Vulnerability Management e Incident Response.

```text
Política de segurança -> Controles técnicos -> Detecção -> Resposta
```

## 🎯 Casos de Uso Comuns
- Implantar modelo de menor privilégio em IAM
- Monitorar ameaças com GuardDuty e Inspector
- Padronizar resposta a incidentes cloud

## 💰 Modelo de Preço
Custos variam por logs, avaliações de segurança e serviços de detecção.

## ⚙️ Comandos CLI Úteis
```bash
aws iam get-account-summary
aws guardduty list-detectors
aws inspector2 list-findings
```
