# Amazon API Gateway

## O que é
Serviço gerenciado para criar, publicar e proteger APIs REST, HTTP e WebSocket.

## Quando usar
- Expor endpoints para Lambda, ECS/EKS ou serviços backend.
- Aplicar autenticação, throttling e versionamento de APIs.

## Pontos de prova
- Integração nativa com Lambda.
- Recursos de segurança com IAM, JWT e planos de uso.

## CLI útil
```bash
aws apigateway get-rest-apis
aws apigatewayv2 get-apis
```
