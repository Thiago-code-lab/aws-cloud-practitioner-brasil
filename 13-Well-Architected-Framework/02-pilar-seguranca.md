# 🔐 Pilar: Segurança

## 📌 O que é?
Garante proteção de workloads, dados e identidades com controles preventivos, detectivos e de resposta.

## 🧠 Analogia Simples
É segurança em camadas de um banco: porta, cofre, câmera e equipe de resposta.

## 🏗️ Como funciona
Princípios-chave: identidade forte, rastreabilidade, segurança em camadas, automação, proteção de dados e preparação para incidentes.

```text
IAM + Logs + Criptografia + Detecção -> Menor superfície de risco
```

## 🎯 Casos de Uso Comuns
- Aplicar least privilege em IAM
- Proteger dados em repouso e trânsito
- Detectar e responder ameaças com GuardDuty

## 💰 Modelo de Preço
Custos de logs, criptografia e serviços de segurança habilitados.

## ⚙️ Comandos CLI Úteis
```bash
aws iam list-roles
aws cloudtrail describe-trails
aws kms list-keys
```
