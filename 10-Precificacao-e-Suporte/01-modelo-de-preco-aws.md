# Modelo de Preço AWS

## O que é
A AWS segue cobrança por consumo, com elasticidade e compromisso opcional para economia.

## Conceitos centrais
- Pague apenas pelo uso efetivo de recursos.
- Desligar recursos não usados é parte essencial de FinOps.
- Reserva e Savings Plans reduzem custo com previsibilidade.

## Pontos de prova
- CapEx x OpEx é tema recorrente.
- Elasticidade reduz superprovisionamento.
