# Amazon Bedrock

## O que é
Serviço gerenciado para construir aplicações de IA generativa usando modelos fundacionais de múltiplos provedores.

## Quando usar
- Criação de assistentes, geração de conteúdo e sumarização.
- Casos de GenAI com governança e integração nativa na AWS.

## Pontos de prova
- Permite escolher modelo sem gerenciar infraestrutura de inferência diretamente.
- Inclui recursos para segurança e controle de uso.

## CLI útil
```bash
aws bedrock list-foundation-models
```
